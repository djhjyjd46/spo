# СтройПромОбразование (SPO) - WordPress Project

Проект веб-сайта для компании "СтройПромОбразование". Построен на базе WordPress с использованием современной верстки на Tailwind CSS.

## 🚀 Технологический стек

- **CMS:** WordPress
- **Стилизация:** Tailwind CSS (v3)
- **Сборка стилей:** PostCSS / Tailwind CLI
- **Frontend скрипты:** Vanilla JS, Glider.js, Swiper
- **Оптимизация:** Terser (для минимизации JS)

## 🛠 Разработка и сборка

Все основные команды выполняются через `npm` из корневой директории проекта.

### Предварительные условия

1. Установленный **Node.js** и **npm**.
2. Локальный сервер (например, **OSPanel**), настроенный на директорию проекта.

### Команды

- **Режим разработки (Watch):**
  Запускает отслеживание изменений в файлах и автоматическую пересборку стилей Tailwind.

  ```bash
  npm run dev
  ```

- **Сборка для продакшена:**
  Генерирует оптимизированный CSS файл.

  ```bash
  npm run build
  ```

- **Минимизация JavaScript:**
  Запускает скрипт для сжатия JS-файлов темы с помощью Terser.
  ```bash
  npm run minify-js
  ```

## 📂 Структура темы

Основная тема находится по пути: `wp-content/themes/mythem/`

- `css/` — Скомпилированные и исходные стили.
- `js/` — Клиентские скрипты.
- `parts/` — Шаблоны частей страниц (header, footer и т.д.).
- `woocommerce/` — Переопределенные шаблоны WooCommerce (если применимо).
