# My Yandex Reviews

Плагин WordPress для создания и отображения отзывов с Яндекс.Карт.

## Возможности

- ✅ Собственный тип записей "Отзывы Яндекс" в админке WordPress
- ✅ Произвольные поля для данных отзыва (автор, рейтинг, дата, ссылка)
- ✅ Шорткод `[yandex_reviews]` для отображения отзывов
- ✅ Красивые адаптивные карточки отзывов
- ✅ Звёздочки рейтинга
- ✅ Поддержка миниатюр (фото клиентов)
- ✅ Автоматическое подключение стилей

## Установка

1. Загрузите папку плагина в `wp-content/plugins/`
2. Активируйте плагин в админке WordPress
3. В меню появится новый раздел "Отзывы Яндекс"

## Использование

### Добавление отзывов

1. Перейдите в "Отзывы Яндекс" → "Добавить новый"
2. Заполните:
   - Заголовок отзыва
   - Текст отзыва в основном редакторе
   - Имя автора
   - Рейтинг (1-5 звёзд)
   - Дата отзыва
   - Ссылка на оригинал на Яндексе
   - Миниатюру (фото клиента) - опционально

### Отображение отзывов

Используйте шорткод на любой странице или в записи:

```
[yandex_reviews]
```

Параметры шорткода:
- `limit` - количество отзывов (по умолчанию: все)
- `order` - порядок сортировки: DESC или ASC (по умолчанию: DESC)

Примеры:
```
[yandex_reviews limit="6"]
[yandex_reviews limit="3" order="ASC"]
```

## Поля отзыва

- **review_author** - имя автора отзыва
- **review_rating** - рейтинг от 1 до 5 звёзд
- **review_date** - дата отзыва (формат: YYYY-MM-DD)
- **review_link** - ссылка на оригинальный отзыв на Яндексе

## Настройка стилей

Стили автоматически подключаются из файла `assets/css/styles.css`. 
Вы можете изменить их под дизайн вашего сайта.

Основные CSS классы:
- `.yandex-reviews-container` - контейнер всех отзывов
- `.yandex-reviews-grid` - сетка карточек
- `.review-card` - отдельная карточка отзыва
- `.review-stars` - контейнер звёздочек
- `.review-link` - ссылка на Яндекс

## Совместимость

- WordPress 5.0+
- PHP 7.4+
- Адаптивный дизайн для всех устройств
