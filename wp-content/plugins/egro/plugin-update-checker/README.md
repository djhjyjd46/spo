# Plugin Update Checker для ArtCly SMTP

Эта упрощенная версия системы обновлений для плагина ArtCly SMTP.

## Как работает

1. Система проверяет наличие обновлений по указанному URL
2. Загружает информацию о последней версии в формате JSON
3. Если версия в файле новее установленной, предлагает обновление

## Формат JSON-файла с информацией об обновлении

```json
{
  "name": "ArtCly Egro",
  "version": "1.0.6",
  "download_url": "https://ваш_сайт/плагины/artcly-smtp.zip",
  "homepage": "https://ваш_сайт/",
  "requires": "5.0",
  "tested": "5.8",
  "last_updated": "2023-11-01",
  "sections": {
    "description": "Описание плагина",
    "changelog": "<h4>1.0.6</h4><p>Улучшения безопасности</p>"
  }
}
```
